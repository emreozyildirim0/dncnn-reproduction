# Datasets

The datasets are not included in this repository. They come from the original
DnCNN repository: https://github.com/cszn/DnCNN

Download and place them here so the folder looks like:

```
data/
  Train400/   400 grayscale training images (180x180)
  Set12/      12 classic test images
  BSD68/      68 images from the Berkeley segmentation dataset
```

- `Train400` is under `TrainingCodes/DnCNN_TrainingCodes_v1.0/data/Train400` in
  the original repository.
- `Set12` and `BSD68` are the standard grayscale test sets used in the paper.

All images are expected to be `.png`.
